## Overview

Roam India is a comprehensive travel recommendation application designed to help tourists and locals discover the best places across India. From serene parks to bustling shopping centers, Roam India serves as your perfect companion for exploring the diverse attractions each Indian state has to offer.

## Features

- **State-wise Exploration**: Browse through all Indian states and their unique attractions
- **Categorized Recommendations**: Find places based on categories:
  - Famous Coffee Shops ☕
  - Restaurants 🍽️
  - Kid-friendly Places 🎡
  - Parks 🌳
  - Shopping Centers 🛍️
- **Detailed Information**: Get comprehensive details about each location
- **Modern UI**: Built with Jetpack Compose for a smooth, native Android experience
- **Splash Screen**: Engaging animation using Lottie

## Technical Stack

- **Language**: Kotlin
- **UI Framework**: Jetpack Compose
- **Navigation**: Jetpack Navigation Component
- **Animations**: Lottie
- **Architecture**: MVVM (Model-View-ViewModel)
